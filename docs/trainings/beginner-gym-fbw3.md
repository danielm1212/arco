# Beginner · Siłownia · Full Body · cykl 3 dni

> **Poziom:** początkujący · **Środowisko:** pełna siłownia · **Sesje:** 3× / tydz. (A → B → C, rotacja) · **Częstotliwość / partia:** ~3× · **RIR:** 2–3 · **Progresja:** double progression
>
> Cel: nauczyć wzorców ruchowych na barbell, zbudować bazę siły i masy, wyrobić nawyk. Compoundy w niższych zakresach (technika + siła), pomocnicze wyżej. Serie robocze bez rozgrzewkowych.

## Notatki

- **Bilans tygodniowy** (serie robocze, partia główna): push ~8 · pull ~8 · nogi ~9 (+ hinge 3) · core ~7. Push:pull ≈ 1:1 — zbalansowane od startu. Łydki obecne (uwaga z audytu zamknięta).
- **Progresja:** trafisz górę zakresu na wszystkich seriach przy RIR ≥1 → +2.5 kg góra / +5 kg dół następnym razem. Na tym poziomie ciężar często rośnie co sesja — to OK.
- **Kolejność:** compound ciężki → pomocniczy → izolacja/core. Nie zmieniaj — stałość napędza overload.
- **2× / tydz. fallback:** jeśli ktoś nie robi 3 sesji, leć A → B naprzemiennie, C wpada w kolejny tydzień.

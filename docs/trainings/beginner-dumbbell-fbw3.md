## Notatki

- **Bilans tygodniowy:** push ~8 · pull ~9 (w tym pion 3) · nogi ~9 (+ hinge 6) · core ~5. Push:pull ≈ 1:1 (naprawione względem 1.43:1 z FBW-hantle). Vertical pull, łydki i pośladki obecne — wszystkie 5 uwag z audytu zamknięte.
- **Vertical pull bez sprzętu:** jeśli user nie ma gumy ani drążka, `DB Pullover` łapie część funkcji lats, ale to niepełny zamiennik — guma to najtańszy fix (~30 zł) i rekomendacja onboardingu.
- **Load ceiling:** przy lekkich hantlach wejdź w górę zakresu i tempo (3–1–1) zanim uznasz stall. Unilateral (split squat, 1-arm row) mnoży dostępny bodziec.
- **2× fallback:** A → B naprzemiennie.

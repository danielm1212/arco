## Progresja (drabinki leverage)

Bodyweight nie dokłada kilogramów — dokłada **trudność wariacji**. Gdy trafisz górę zakresu na wszystkich seriach, wejdź o oczko wyżej:

- **Squat:** BW Squat → Tempo Squat (3s dół) → Split Squat → Bulgarian → Pistol-assisted → Pistol
- **Horizontal push:** Incline Push-up → Push-up → Feet-elevated → Archer → Pseudo-planche → One-arm progression
- **Vertical push:** Pike Push-up → Feet-elevated Pike → Wall-assisted HSPU → HSPU
- **Vertical pull:** Negatives → Band-assisted Pull-up → Pull-up → Tempo → Weighted (plecak)
- **Horizontal pull:** Incline Inverted Row → Poziomy → Feet-elevated → Archer Row

**0 sprzętu:** dokładaj plecak z książkami do pull/row i split squatów, gdy wyczerpiesz zakres powtórzeń.

## Notatki

- **Vertical pull to pięta achillesowa bodyweight bez drążka.** Towel/backpack rows łapią głównie horizontal. Drążek rozporowy (~50 zł) to rekomendacja onboardingu dla kogokolwiek na serio.
- **Bilans:** push mocny (natura kalisteniki), pull podbity świadomie przez 2× vertical + inverted row, żeby nie wpaść w klasyczny push-heavy dryf.
- RIR 2–3, ostatnia seria push-upów/rows może iść bliżej upadku.

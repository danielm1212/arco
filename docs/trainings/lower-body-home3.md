# Beginner–Intermediate · Dom z hantlami · Pośladki i nogi · cykl 3 dni

> **Poziom:** początkujący–średniozaawansowany · **Środowisko:** dom z hantlami · **Sesje:** 2–3× / tydz. (A → B → C) · **RIR:** 2–3 · **Progresja:** double progression
>

## Zasady

- Trenuj według kolejności A → B → C, także przy dwóch sesjach tygodniowo.
- Bez ławki wykonuj glute bridge i wyciskanie hantli na podłodze.
- Gdy hantle są za lekkie, najpierw zwiększ powtórzenia, a potem zwolnij opuszczanie do 3 sekund.
- Po osiągnięciu górnego zakresu we wszystkich seriach zwiększ ciężar.
